服务网格 FSM（Flomesh Service Mesh）

Flomesh服务网格（FSM）是羽量级的、易拓展的、可定制的服务网格软件；提供开箱即用的服务网格能力，同时提供丰富接口满足DIY场景需求

低资源高性能易扩展简单易用更安全

低资源

数据平面sidecar起步使用低于10M的内存，适合高密度部署，并且具有更好的成本结构。sidecar可执行文件和容器镜像均低于10M，满足云原生时代的“快速部署、快速回收”的需求

高性能

在cgroup CPU=1000m的情况下，提供了行业内最大的吞吐量TPS，最小的请求处理延迟，最丰富的指标采集能力；提供了metrics+logging+tracing三位一体的数据采集和处理能力

易扩展

在sidecar proxy层面提供C++，JS，REST Admin三层的扩展接口和能力。在控制平面提供了CRD+Controller的扩展模型。在用户体验层提供了GUI、REST、CLI扩展能力。提供了多方位、全角度的DIY服务网格的能力

简单易用

具备k8s管理能力的团队可以快速的部署和实施FSM，不需要学习和理解更多的配置文件

更安全

FSM支持非iptables的流量拦截方案，无需特权用户。使得服务进程满足多租户环境下的安全需求
